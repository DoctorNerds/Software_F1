*COMANDOS IMPORTANTES NO TERMINAL*

Para criar um ambiente virtual
python -m venv venv

Para ativar o ambiente virtual (venv):
./venv/Scripts/Activate.ps1

Para instalar o streamlit:
pip install streamlit

Para rodar arquivo com streamlit (caminho: D:\FastF1\data_viz.py)
python -m streamlit run "D:\FastF1\data_viz.py"
